## 项目布局
```
|-- images               // 公共图片
|-- lib                  // 公共文件
|-- pages                // 页面
|   |-- index            // 首页
|   |-- list             // 商品列表
|   |-- details	         // 商品详情
|   |-- user             // 个人中心
|   |-- order            // 我的订单
|   |-- coupon           // 优惠券
|   |-- cart             // 购物车
|   |-- addressList      // 我的地址
|   |-- addressAdd       // 添加地址
|-- README.md            // 说明
```
